# PPG-single-modal-TF-rUnet-MP-DANN
PPG single modal TF-rUnet MP-DANN

# PPG-based TF-rUnet used for Emotion Recognition

## Description

This repository contains two main components:

1. **SSL Pre-training Data Preparation** — Methods for preparing data for Self-Supervised Learning (SSL) pre-training.
2. **Model Architectures** — All model structures used in this work.

## Contents

### 1. signal_transformation

This section provides implementations of **12 signal transformations** used for data augmentation and representation learning.

### 2. tf-runet

This section contains the source code for the **TF-rUnet model architecture**.

### 3. Pre-trained Weights

Pre-trained weights are **available for research purposes only**.

To request access, please send an email to **jmchen22@m.fudan.edu.cn** with the following subject line format: 

> PPG TF-rUnet SSL weight - [Full Name] - [Institution]

**Note:** Please replace `[Full Name]` with your actual full name and `[Institution]` with your affiliated institution.
